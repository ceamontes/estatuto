# Estatuto

## Comandos e ambientes definidos em `estatuto.cls`

- `\titulo{#1}`

	Título do documento.

- `\section{#1}`

	Marcação de capítulos.

- ```tex
	\begin{artigos}
		\item #1
	\end{artigos}
	```

	Marcação de artigos.

- ```tex
	\begin{itens}
		\item #1
	\end{itens}
	```

	Marcação de itens.

- ```tex
	\begin{paragrafos}
		\item #1
	\end{paragrafos}
	```

	Marcação de parágrafos.

- `\paragrafo{#1}`

	Marcação de parágrafo único.

- `\assinaturas`

	Espaço para assinaturas.
